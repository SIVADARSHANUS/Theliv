# Theliv — Design Context Assistant

A RAG-powered tool that converts messy client inputs into a clear, structured design direction report before the first pixel is placed.

## Links

- **Prototype:** https://theliv.base44.app/
- **Case Study / Portfolio:** https://www.notion.so/sivadarshanus/Theliv-From-messy-client-inputs-to-clear-design-briefs-340afdb036d780d8b864d82ad98ead35

## Problem

Web designers consistently start projects with incomplete, contradictory, or misunderstood client inputs. Discovery has no structure. Clients use vague language. References arrive with no annotation. The result is 3 to 5 revision cycles that burn time on both sides.

## Solution

Theliv stores client inputs, analyzes reference websites, detects conflicts, and generates a structured 6-section design direction report. It is decision-support infrastructure, not a design generator.

## Validated Through

- 20+ designer surveys
- 5 in-depth designer interviews
- Identified 10 to 25 hours of rework waste per project

## Built With

- RAG architecture
- Prompt engineering
- Vibe coding
- Vision model and OCR for reference extraction
- Base44
